# ワイヤーフレーム - ワイン発注管理SaaS

作成日: 2026-04-02
担当: UI/UXデザイン
形式: テキスト / ASCII

凡例:
- `[ ]` ボタン
- `( )` 入力フィールド
- `|...|` カード・コンテナ
- `[x]` アイコン付き要素
- `===` 区切り線

# ワイン発注管理SaaS MVP タスク整理

作成日: 2026-04-02

---

## プロジェクト概要

- **目的**: 酒屋（管理者）と飲食店約80名（発注者）をつなぐ発注管理SaaS
- **技術スタック**: Next.js / Supabase / Vercel
- **方針**: MVP優先

---

## MVP スコープ定義

### ユーザー種別
1. **管理者（酒屋）**: 商品管理・発注受付・在庫管理
2. **発注者（飲食店）**: 商品閲覧・発注・発注履歴確認

### MVP に含める機能
- [ ] 認証（管理者・発注者のロール分離）
- [ ] 商品一覧・詳細表示
- [ ] 発注機能（カートで複数商品を一括発注）
- [ ] 発注履歴確認（発注者側）
- [ ] 発注管理（管理者側：受付・ステータス更新）
- [ ] 商品管理（管理者側：登録・編集・削除）

### MVP から除外（将来対応）
- 決済連携
- 在庫自動管理
- メール通知
- 分析ダッシュボード
